Remote Sensing Web Application

Remote Sensing, is a web application oriented to the processing of hyperspectral images (HSI) using Deep Learning (DL) modeles.

The application allows to process some of the most known and commonly used HSI by the research community. Through a form, the user has the ability to execute over a selected image, previously trained DL models oriented to dimensional reduction, feature extraction and information extraction.
