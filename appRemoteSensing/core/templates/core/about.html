Remote Sensing Web Application

Remote Sensing, es una aplicación web orientada al procesamiento de imágenes hiperespectrales (HSI) por medio de técnicas de Deep Learning (DL).

La aplicación permite procesar algunas de las HSI más conocidas y comúnmente utilizadas por la comunidad investigativa. A través de un formulario, el usuario tiene la capacidad de ejecutar sobre una imagen seleccionada, modelos previamente entrenados de DL orientados a reducción dimensional, extracción de características y extracción de información.
